# Obsidian Empty Vault

This is my personal [Obsidian](https://obsidian.md/) vault template.  It is based on Steph Ango personal [vault](https://stephango.com/vault). I took the parts I liked and tailored them to my needs.

## Setup

1. Create an empty vault
2. Install the [TaskNotes](https://tasknotes.dev/) plugin
3. Download `obisidan-empty-vault` and copy the contents in your empty vault

## Hotkeys

- ⌘ N - `QuickAdd: All Templates` (Use to create new notes using templates)
- ⌘ I - `Templates: Insert template` (Merges templates into current note)
- ⌘ ' - `Toggle Live Preview/Source mode`
- ⌘ T - `TaskNotes: Create new task`
- ⌘ M - `Move current file to another folder`
- ⌘ O - `Omnisearch: Vault search`
